# GuidewireAI — Accelerator Slide Packs

Client-ready decks explaining all seven AI accelerators: **purpose**, **worked example**, **demo preview**, and **productionalisation**.

## Formats

| Style | File | Regenerate |
|---|---|---|
| **McKinsey** (recommended) | [`accelerators-slide-pack.pptx`](accelerators-slide-pack.pptx) | `python scripts/generate-slide-pack.py` |
| **NTT Global Insurance template** | [`guidewire-ai-accelerators-slide-pack.pptx`](guidewire-ai-accelerators-slide-pack.pptx) | `python scripts/generate-guidewire-slide-pack.py` |
| **HTML** (browser) | [`accelerators-slide-pack.html`](accelerators-slide-pack.html) | — |

```bash
pip install python-pptx pillow
npm run slides              # McKinsey pack (default)
npm run slides:ntt          # NTT DATA 2025 template pack
```

## McKinsey pack contents (18 slides)

1. Title — GuidewireAI SDLC Test Accelerators
2. Executive summary (challenge / response / metrics)
3. Lifecycle architecture + accelerator table
4. Section — The seven AI accelerators
5–11. **One slide per accelerator** — purpose · example · productionalise · demo image/GIF · ROI card
12. Section — Productionalise the suite
13. Productionalisation roadmap (Phase 0–3)
14. Katalon companion asset
15. Technical architecture (reference material + RAG)
16. Value at a glance (ROI + first production step)
17. Adoption path
18. Next steps

## Each accelerator slide includes

| Block | Content |
|---|---|
| **Action title** | McKinsey-style headline (navy header bar) |
| **So what** | Teal-bar insight strip |
| **Purpose** | Problem solved |
| **Example** | Input → output walkthrough |
| **Productionalise** | 3 concrete pilot → enterprise steps |
| **Demo** | PNG screenshot or GIF (`docs/media/`, `public/media/`) |
| **ROI card** | Indicative metric from catalog |

## Demo assets

- `docs/media/*-demo.png` — generated by `scripts/generate-accelerator-demos.py`
- `public/media/code-review.gif`, `defect-triage.gif` — animated (preferred in deck when present)
